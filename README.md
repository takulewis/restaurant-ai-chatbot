# 🍽️ Restaurant AI Chatbot

An AI-powered restaurant chatbot that provides customers with instant access to menu information, operating hours, reservations, promotions, and customer support services 24/7.

## Features

- 24/7 customer assistance
- Interactive AI-powered conversations
- Menu information and recommendations
- Restaurant operating hours
- Location and contact details
- Frequently Asked Questions (FAQs)
- Reservation and booking support
- Fast and reliable customer service

## Project Objective

This project aims to improve customer experience and engagement by providing an intelligent virtual assistant that can answer customer inquiries and provide restaurant information at any time.

## Technologies Used

- Python
- OpenAI API
- Flask/FastAPI
- HTML, CSS, JavaScript
- REST APIs

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/restaurant-ai-chatbot.git
   ```

2. Navigate to the project directory:
   ```bash
   cd restaurant-ai-chatbot
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Configure your environment variables:
   ```env
   OPENAI_API_KEY=your_api_key
   ```

5. Run the application:
   ```bash
   python app.py
   ```
